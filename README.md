# Hansa Electrical - Billing & Point Estimation Portal ⚡

A comprehensive, responsive React & Vite dashboard application designed specifically for electrical wiremen, contractors, and technicians. It streamlines business operations by combining point-based cost estimation, professional billing, client relationship management, and inventory tracking into a single unified platform.

## 🚀 Key Features

*   **📊 Interactive Dashboard**: Real-time business analytics displaying total earnings, outstanding invoices, active quotations, print history, and quick access shortcuts.
*   **🔌 Point Estimator**: Quickly estimate labor and materials cost based on electrical points (e.g., socket points, fan fittings, camera installations) and convert estimates directly into formal quotations with a single click.
*   **🧾 Invoice & Quotation Generator**:
    *   Create professional GST-compliant invoices and quotations.
    *   Calculate CGST, SGST, IGST, taxable amounts, round-offs, and automatically convert numerical totals to words.
    *   Include customizable terms and conditions.
*   **🖨️ Print-Ready Layouts**: Beautiful, clean, print-optimized document designs optimized to save as PDF or print directly, featuring dynamic document naming.
*   **👥 Client Management**: Built-in client ledger to quickly store addresses, GSTIN numbers, and contact details.
*   **📦 Inventory & Material Database**: Track your materials catalog, standard units (NOS, MTR, etc.), and default rates to instantly populate invoices.
*   **💾 Local Backup & Restore**: Full JSON-based import/export system to easily backup data, transfer between devices, or restore settings.
*   **🔒 Local-First Storage**: Securely stores all invoices, quotations, clients, and inventory data locally using the browser's `localStorage` (works entirely offline!).

## 🛠️ Technology Stack

*   **Frontend Library**: React 19
*   **Build Tool**: Vite 8
*   **Icons**: Lucide React
*   **Styling**: Pure CSS3 Custom Variables (featuring dark-mode compatibility and a clean, responsive layout)
*   **Code Quality**: Oxlint
